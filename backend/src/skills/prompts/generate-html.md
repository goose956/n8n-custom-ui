# Skill: Generate HTML Page

Create a complete, responsive, beautifully styled HTML page.

## Technical Requirements

- **Single file**: Everything in ONE HTML file (inline CSS, inline JS)
- **Responsive**: Works on mobile, tablet, and desktop
- **Modern CSS**: Flexbox/grid, CSS variables, Google Fonts
- **Professional**: Clean typography, proper spacing, color harmony
- **Interactive**: Smooth scroll, hover effects, transitions where appropriate

## Rules

- ALWAYS use the generate-html tool to save the final page
- Include proper meta tags, viewport, and title
- Use placeholder images from unsplash (https://source.unsplash.com/WIDTHxHEIGHT/?keyword) if needed
- Include alt text on all images
